## Project Description

RevSpace is a full-stack web application that allows users to register for an account and engage in
a social media platform. Users can create, view, and like posts, as well as comment on them. The
application also provides the functionality to register, edit, and view user profiles.

## Technologies Used

- Frontend: HTML, CSS, TypeScript, Angular 4
- Frontend Testing: Selenium with Java, Cucumber BDD
- Backend: Java 8, Maven, Spring Boot, Spring Data JPA, Spring MVC
- Backend Testing: Spring Test, JUnit 4, Mockito, Postman
- DevOps and CI: AWS S3 (frontend deployment), AWS EC2 (backend deployment), Jenkins, GitHub
  Webhooks
- Database and Other Tools: PostgreSQL, Log4J, IntelliJ, VS Code, DBeaver, GitHub, Trello

## Your Role

As a team lead for the RevSpace project, my responsibilities included leading the development of the
