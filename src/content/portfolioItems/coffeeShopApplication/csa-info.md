## Project Description

Coffee Shop Application is a full-stack web application that allows customers to order various items
typically found in a coffee shop. Customers can create accounts, log in, and place orders. Store
managers can track and update order status. The application also includes features like order
history, daily specials, and contact information.

## Features

- Customer account creation and login
- Ordering items with ingredient customization
- Dynamic daily specials
- Order history for customers
- Cart review and payment overview
- Order tracking for store managers
- About page and Contact Us functionality

## Technologies Used

- Frontend: HTML, CSS, TypeScript, Angular
- Frontend Testing: Selenium with Java, Cucumber BDD
- Backend: Java 8, Maven, Spring Boot, Spring Data JPA, Spring MVC
- Backend Testing: Spring Test, JUnit, Mockito, Postman
- Database and Other Tools: PostgreSQL, IntelliJ, VS Code, DBeaver, GitLab, Trello
